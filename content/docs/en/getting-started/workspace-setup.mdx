# Workspace Setup

Before using NOGL day-to-day, a workspace admin should complete these one-time setup steps.

## Adding team members

1. Go to **Settings → Team**.
2. Click **Invite member**.
3. Enter the email address and select a role:
   - **Admin** — full access including billing and integrations
   - **User** — can view and edit catalog, rules, and reports
   - **Viewer** — read-only access to all data
4. Click **Send invite**. The invitee will receive an email with a sign-in link.

## Connecting your catalog

NOGL imports your product catalog from your e-commerce platform or via CSV.

### CSV import

1. Go to **Settings → Data Sources → Catalog Import**.
2. Download the CSV template.
3. Fill in your SKUs, titles, prices, and costs.
4. Upload the completed file.

### Platform integration

Supported platforms: Shopify, WooCommerce, Magento. See [Catalog Import](/en/fractional-cfo/docs/admin-integrations/catalog-import) for step-by-step instructions.

## Currency and locale

1. Go to **Settings → Workspace**.
2. Set your default currency (e.g. EUR) and timezone.
3. These defaults apply to all reports and price displays.

## Notification preferences

Each user can set their own notification preferences at **Settings → Notifications**. Workspace admins can set platform-wide defaults.
