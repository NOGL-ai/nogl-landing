# Release Notes

## v1.7 — April 2025

### New features

- **In-app help center** — Documentation is now searchable directly from within the app (⌘K → type your question).
- **Demand shaping campaigns** — Flash sale and bundle discount tools are now available under Repricing.
- **Webhook events** — Stream repricing and alert events to any HTTP endpoint.
- **Asset bulk upload** — Upload images for multiple products at once using a SKU-named ZIP file.

### Improvements

- Repricing history now shows which competitor triggered each rule change.
- Replenishment recommendations now account for pending purchase orders.
- Dark mode is now fully supported across all pages.

### Bug fixes

- Fixed: competitor price history chart sometimes displayed incorrect dates after DST change.
- Fixed: CSV export for large catalogs (>10,000 SKUs) no longer times out.

---

## v1.6 — February 2025

### New features

- **Tiered pricing** — Set different prices per quantity bracket.
- **Competitor discovery** — NOGL now suggests competitor URLs based on your product titles.
- **Alert digest** — Daily email summarizing all alerts instead of one email per alert.

### Improvements

- Forecasting model now incorporates weekend sales patterns more accurately.
- Product search in catalog is now full-text across title, SKU, description, and brand.

---

## v1.5 — November 2024

### New features

- **Auto repricing rules** — First release of the rule engine.
- **Competitor tracking** — Track unlimited competitor websites on Growth and Enterprise plans.
- **Replenishment module** — Reorder recommendations with configurable safety stock.
